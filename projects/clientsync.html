ClientSync

A SaaS platform that integrates businesses with HubSpot CRM, enabling real-time lead synchronization, pipeline tracking, and automation without switching tools.

Problem

Small to mid-sized teams often struggle with keeping their internal tools and HubSpot CRM in sync, leading to lost leads, duplicated data, and inefficient workflows.

Solution

ClientSync acts as a bridge between internal tools and HubSpot, offering a unified dashboard for managing contacts, deals, and pipelines. Real-time webhooks ensure CRM data stays updated across platforms.

Tech Stack

  • Backend
    Laravel
  • Frontend
    Vue.js
  • State Management
    Pinia
  • Database
    MySQL

My Role

  1. Integrated secure OAuth2 authentication with HubSpot.
  2. Designed React-based UI with Redux for state management.
  3. Built Laravel API routes for CRM data fetching, updating, and syncing.
