Social Scheduler

Social Scheduler is a full-stack application that helps users plan, schedule, and manage social media posts across multiple platforms from a single dashboard. It gives creators, businesses, and agencies a way to save time, maintain consistency, and analyze their content performance without juggling multiple apps.

Problem

Managing multiple social media accounts manually is time-consuming and inconsistent, with businesses struggling to post at the right times, track performance, and coordinate content across different platforms.

Solution

Social Scheduler provides a unified dashboard where users can create, schedule, and automate posts across platforms, store reusable content, and access analytics—all in one place—making social media management faster, more consistent, and more collaborative.

Tech Stack

  • Backend
    Laravel
  • Frontend
    React
  • State Management
    Redux
  • Database
    MySQL

My Role

  1. Designed and built the full-stack architecture using React (Redux) on the frontend and Laravel APIs on the backend.
  2. Developed the drag-and-drop content calendar, post composer, and reusable media library.
  3. Integrated major social media APIs (Facebook/Instagram, Twitter/X, LinkedIn, TikTok) for publishing and analytics.
  4. Implemented secure authentication and role-based access control for teams and collaborators.
